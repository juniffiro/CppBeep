# Cpp-Beep
One of the interesting things you can add to your project, such as DLL, when you successfully inject it.
<br/>
```C++
BOOL Beep(
  DWORD dwFrequency,
  DWORD dwDuration
);
```
**dwFrequency**
<br/>
The frequency of the sound, in hertz (Hz).

**dwDuration**
<br/>
The duration of the sound, in milliseconds.
